# hardware

This directory stores notes and design artifacts for the real-hardware implementation.

The project is currently simulation-first, but all simulation designs should remain transferable to a realistic hardware solution whenever possible.

Suggested future contents:

- `pin-map.md`: ESP32 pin definitions
- `sensor-interface.md`: DS18B20 wiring and power notes
- `pwm-driver.md`: PWM output and low-side MOSFET driving notes
- schematics, wiring diagrams, BOMs, and component selection records
